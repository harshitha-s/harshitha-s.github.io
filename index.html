Array Search

searching an element in an array

Follow me onGitHub

INTRODUCTION

The program searches and reports if a given element is present in the array or not

WORKING

  • The number of elements are entered.
  • The elements are given as input in random order.
  • The elements are then sorted in ascending order.
  • The search element is given as input.
  • The element is searched in an array using binary search.
  • Results are displayed.
